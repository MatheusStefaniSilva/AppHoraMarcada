# Hora Marcada

Projeto em desenvolvimento para criar uma plataforma de agendamento online para prestadores de serviço.

## Onde estamos

A base do app já está em funcionamento com Next.js, incluindo uma interface inicial de agendamento com fluxo de wizard, seleção de serviços, datas e horários. Neste momento, o foco está na estrutura visual e no fluxo principal do usuário.

## O que faremos no futuro

A próxima etapa é transformar essa versão inicial em um MVP completo, com:

- autenticação para prestadores;
- dashboard para gerenciar agenda, serviços e horários;
- integração com banco de dados e persistência real;
- envio de e-mails e confirmação de agendamentos;
- implantação para uso real.

## Como rodar

```bash
npm install
npm run dev
```

A aplicação ficará disponível em http://localhost:3000.
